*ngFor

Usamos "*ngFor" para iterar por el array "animales" el cual tiene tres objetos con las propiedades "tipo", "nombre", "edad", para ello escribimos "let animal of animales". Tambien a modo opcional podemos obetener el index de cada uno con "index as i".

# TIPO NOMBRE EDAD
{{i+1}} {{animal.tipo}} {{animal.nombre}} {{animal.edad}} 

  <table class="table">
    <thead>
      <tr>
        <th scope="col">#</th>

        <th scope="col">NOMBRE</th>
        <th scope="col">EDAD</th>

    <tbody>
      <tr *ngFor="let animal of animales; index as i">
        <th scope="row">i+1</th>
        <td>animal.tipo</td>
        <td>animal.nombre</td>
        <td>animal.edad</td>
      </tr>
    </tbody>
  </table>
